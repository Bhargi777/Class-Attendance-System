# Classroom Attendance Tracker

A minimal, interactive seating-chart-based attendance tracker for classrooms. Click on a seat to mark a student absent or present. Built with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies.

## Features

- **Interactive seating chart** — tap any seat circle to toggle present / absent
- **Live stats** — see present, absent, and total counts update in real time
- **Progress bar** — visual indicator of attendance percentage
- **Swap layout** — flip left and right column sides
- **Copy report** — one click to copy a formatted attendance report to your clipboard
- **Responsive** — works on desktops, tablets, and phones

## Usage

Open `index.html` in any browser. That's it.

- Click a **green circle** to mark the student absent (turns red)
- Click a **red circle** to mark them present again (turns green)
- Use **Swap sides** to mirror the seating layout
- Use **Copy report** to get a text summary on your clipboard
